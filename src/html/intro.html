A pegada de carbono individual é uma medida essencial para entender e minimizar o impacto ambiental das nossas atividades diárias.

Uma breve introdução

O presente projeto foi pensado pelos alunos João Pedro Silva, Daniel Mantilha, Derick, João Pinheiro e Gabriel, como o objetivo de criar um site com o tema “Créditos de Carbono”. Tal objetivo foi concretizado por meio de diversas páginas, incluindo uma que explica o cálculo e uma que é usada para realizar o cálculo das emissões de carbono das “pessoas físicas”.

Para realizar o dito projeto foram pensados os seguintes pontos para explicar o projeto para quem está lendo :

  1. Realizar a explicação da lógica de programação idealizada pelos programadores.
  2. Explicação de como se realiza o cálculo das emissões de carbono com foco primordial para as pessoas físicas.
  3. Indicar outros sites que podem ser usados pra o cálculo focado nas empresas

O que é a pegada de carbono ?

A pegada de carbono é de forma resumida a emissão de carbono pelas mais variadas atividades desenvolvidas pelos seres humanos, como por exemplo: o gasto com energia elétrica, quantas pessoas vivem na casa e o tipo de combustível do automóvel utilizado.

A pegada de carbono envolve o cálculo das emissões de carbono e a partir do resultado procurar formas de diminuir as suas emissões para assim contribuir com o meio ambiente e diminuir o impacto de fenómenos naturais, como por exemplo: Ondas de calor e chuvas ácidas.

Como diminuir as emissões de CO2?

  1. Reduzir o uso de automóveis, preferindo os veículos de trasporte público, como ónibus e veículos que não poluem, como bicicletas
  2. Reciclar os produtos.
  3. Consumir menos latiicneos e carnes.
  4. Economizar energia e utilizar fontes de energia sustentáveis.
  5. Reduzir o uso de plástico
